# RSM Presentation HTML Template

HTML/CSS translation of the attached `Presentation.pptx` template.

## Included layouts

- title slide with campus hero image
- alternate title slide with city hero image
- text slide
- section divider slide
- dark closure/contact slide
- alternate closure/contact slide

## Assets

Browser-ready brand assets and images extracted from the PowerPoint live in `assets/`.

## Usage

1. Open `index.html` in a browser.
2. Replace the placeholder text in each slide.
3. Swap hero images or contact details as needed.
4. Print to PDF if you need static slide output from the HTML version.
