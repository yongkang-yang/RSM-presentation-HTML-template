# RSM Presentation HTML Template

This repository contains an HTML/CSS presentation template derived from the
Rotterdam School of Management, Erasmus University Microsoft PowerPoint
presentation template.

## Notice

The underlying RSM presentation design and branding are protected. Use this
template only if you are an employee or student of Rotterdam School of
Management, Erasmus University, or otherwise authorized to use the original RSM
presentation template and assets.

## Included layouts

- title slide with campus hero image
- alternate title slide with city hero image
- content slide
- section divider slide
- thank-you slide

## Assets

Browser-ready brand assets and images extracted from the source PowerPoint are
stored in `assets/`.

## Usage

1. Open `index.html` in a browser.
2. Replace the placeholder text in each slide.
3. Swap hero images, contact details, or authorized brand assets as needed.
4. Print to PDF if you need static slide output from the HTML version.
